# Chrome Web Store Checklist

## Package
- Zip the contents of `extension/` so `manifest.json` is at the root of the zip.
- Example output: `tab-gallery-extension.zip`

## Required Assets
- Icon: 128x128 PNG (already in `icons/icon-128.png`).
- Screenshots: at least 1 (1280x800 recommended).
- Optional: promo tile images if you want to use them.

## Listing Text
- Title: Tab Gallery
- Short description: see `STORE_DESCRIPTION.txt`
- Detailed description: see `STORE_DESCRIPTION.txt`
- Category: Productivity (recommended)

## Privacy
- Provide a Privacy Policy URL (host `PRIVACY_POLICY.md` somewhere public).
- Declare data usage: local-only, no data sold or transferred.

## Submission
- Upload zip in the Developer Dashboard.
- Complete all required disclosures.
- Submit for review.
